# HR Attrition Analysis | SQL & Power BI 📊

## 🎯 Project Overview
This project analyzes employee turnover to identify key factors contributing to attrition. By leveraging data visualization, I aimed to provide actionable insights for HR departments to improve retention strategies.

## ⚙️ Data Workflow
* **Data Sourcing:** Extracted raw HR datasets directly from **SQL** databases using optimized queries.
* **Integration:** Seamlessly connected the SQL data to **Power BI** for real-time analysis.
* **Transformation:** Performed data cleaning and modeling within Power BI to ensure accuracy in attrition metrics.

## 🔍 Key Insights
* **Demographics:** Analyzed attrition rates across different age groups and genders.
* **Job Roles:** Identified departments with the highest turnover rates.
* **Satisfaction Levels:** Correlated job satisfaction scores with employee departures.

## 🛠️ Tools Used
* **SQL:** Data extraction and preliminary filtering.
* **Power BI:** Data modeling and interactive dashboard design.
